OpenClaw deployment preset

Run OpenClaw like you mean it.

RunClawd is a curated Docker Compose preset repository for OpenClaw. It gives you an opinionated, production-oriented deployment with a one-line installer — so you can spend less time wiring infra and more time building.

Docker Compose Caddy reverse proxy Optional Cloudflare Tunnel Hardened Docker API via socket proxy Persistent data volume
View repo One-line install What you get

Opinionated defaults

Ships a ready-to-run stack: gateway, web terminal, reverse proxy, optional public tunnel, and persistence.

Production-minded

Avoids mounting Docker socket directly into the app; uses a Docker socket proxy with a limited API surface.

Fast onboarding

Installer prints the onboarding URL, gateway token, terminal password, and tunnel URL when ready.

Install

Linux host. Run as root.
curl -fsSL https://get.runclawd.sh -o runclawd.sh
sudo bash runclawd.sh

What you get

A Compose stack that runs:
- OpenClaw Gateway (runclawd container)
- Web terminal (ttyd)
- Caddy reverse proxy routes: /, /term/, /openclaw/
- Optional ephemeral public access via Cloudflare Tunnel
- Persistent volume for OpenClaw state and workspace
